Panduan Penggunaan

  1. Isi Data Sekolah: Masukkan nama sekolah Anda pada kolom yang tersedia.
  2. Atur Kelas & Target: Pilih kelas awal dan jumlah DPL yang ingin dikuatkan per semester (Standard: 2).
  3. Input Data Awal: Centang dimensi (DPL) yang sudah atau akan dilaksanakan pada tahun pertama (Semester 1 & 2).
  4. Generate: Klik tombol "Generate Roadmap". Sistem akan otomatis merencanakan dimensi untuk kelas-kelas berikutnya.
  5. Cari Data Online: Jika Anda pernah menyimpan data sebelumnya, klik tombol ini dan masukkan nama sekolah untuk memuat ulang data.
  6. Simpan Gambar: Klik tombol "Simpan Gambar" untuk mengunduh infografis visual roadmap.
  7. Simpan Online: Klik tombol "Simpan Online" untuk mengirim data ke database pusat.
